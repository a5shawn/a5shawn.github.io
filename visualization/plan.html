可视化工程师学习路线，分为 4 个阶段，每阶段包含学习目标、核心知识点、推荐资源（附链接）、实战项目建议和时间规划。

🎯 总体目标 ​

在 12 个月 内，掌握 WebGL 原理 + Three.js 实战 + 3D 可视化工程能力，具备应聘「WebGL 前端工程师」或「3D 可视化开发」岗位的能力。

🔰 阶段一：基础准备（第 1-2 月） ​

✅ 学习目标 ​

  • 打牢前端与图形学基础
  • 理解 3D 数学基本概念
  • 能用 Canvas 2D 绘制简单动画

📚 核心内容 ​

主题学习要点推荐资源
HTML5 Canvas 2Dcanvas 元素、2D 上下文、路径绘制、动画循环（requestAnimationFrame）MDN Canvas 教程
JavaScript 进阶ES6+、TypedArray、模块化、异步编程现代 JavaScript 教程（中文）
线性代数基础向量、矩阵、点积/叉积、坐标变换（平移/旋转/缩放）3Blue1Brown《线性代数的本质》（B 站）可汗学院线性代数（英文）
3D 图形基础概念坐标系（世界/相机/屏幕）、投影（透视/正交）、顶点、片段、光照模型《3D 数学基础：图形与游戏开发》第 1–4 章（可选读）

💡 小项目实践 ​

  • 用 Canvas 实现一个会动的钟表
  • 制作一个可拖拽的粒子系统（2D）

🔧 阶段二：WebGL 原理入门（第 3-6 月） ​

目标：理解 WebGL 渲染管线，能手写原生 WebGL 程序

📚 核心内容 ​

主题关键知识点
WebGL 上下文getContext('webgl')、检查支持
着色器（Shader）GLSL 语法、顶点着色器 vs 片元着色器
缓冲区（Buffer）ARRAY_BUFFER、ELEMENT_ARRAY_BUFFER
属性 & Uniform从 JS 传数据到 GPU
MVP 矩阵模型（Model）、视图（View）、投影（Projection）矩阵
纹理映射加载图片、绑定纹理、采样器

🔗 推荐资源（必学） ​

  • ✅ WebGL Fundamentals（中文版）
    👉 最清晰、最系统的 WebGL 入门教程，从“画一个三角形”讲到高级技巧。
  • MDN WebGL 教程
  • 书籍：《WebGL 编程指南》（人民邮电出版社）——配套代码丰富

💡 实战项目 ​

  1. 绘制彩色三角形 → 旋转立方体
  2. 实现带纹理贴图的 3D 立方体
  3. 添加鼠标控制相机视角（使用 mat4.lookAt）

✨ 提示：不要跳过原生 WebGL！它是理解 Three.js 底层的关键。

🚀 阶段三：Three.js 工程化开发（第 7-10 月） ​

目标：熟练使用 Three.js 快速构建 3D 可视化应用

📚 核心内容 ​

模块学习重点
场景（Scene）对象树结构、Group 管理
相机（Camera）PerspectiveCamera / OrthographicCamera
渲染器（Renderer）WebGLRenderer、抗锯齿、阴影
几何体 & 材质BoxGeometry、MeshStandardMaterial、PBR
光照DirectionalLight、PointLight、环境光
动画Clock + requestAnimationFrame
交互OrbitControls、Raycaster（射线拾取）
模型加载GLTFLoader（推荐）、OBJLoader
后处理EffectComposer、Bloom、SSAO

🔗 推荐资源 ​

  • ✅ Three.js 官方文档（含示例）
  • ✅ Three.js Journey（付费但极推荐）
    👉 Bruno Simon 的课程，项目驱动，涵盖 Shader、性能优化、3D 打包等
  • 免费替代：Three.js 中文教程（B 站 - 阿发你好）
  • Three.js Examples —— 查阅官方 Demo 源码

💡 实战项目 ​

  1. 3D 地球仪（带国家标注 + 鼠标悬停）
  2. 产品 3D 展示页（支持旋转、缩放、切换材质）
  3. 3D 数据可视化大屏：用柱状图/散点图在 3D 空间展示销售数据（结合 ECharts-GL 或自定义几何体）

🌐 阶段四：可视化专项 + 工程进阶（持续提升）（第 11-12 月） ​

目标：成为能落地企业级 3D 可视化项目的工程师

📚 重点方向 ​

方向技术栈学习建议
3D 数据可视化ECharts-GL、Three.js + D3、自定义着色器学习如何将业务数据映射到 3D 空间（如热力图、关系网络）
性能优化实例化渲染（Instancing）、LOD、Frustum Culling、Web Worker 分帧阅读 Three.js 性能指南
Shader 进阶自定义材质、后处理、粒子系统学习 The Book of Shaders（中文）
数字孪生/智慧园区CesiumJS（地理空间）、3D Tiles、BIM 模型集成了解 Cesium 官网
工具链Vite + TypeScript + Webpack、GLTF 模型优化、CI/CD 部署使用 Three.js + Vite 模板

🔗 行业资源 ​

  • 招聘要求参考：WebGL 前端工程师招聘要求（猎聘）（见知识库 [15]）
  • 开源项目学习：
    • Three.js 官方 Examples
    • React Three Fiber（R3F）（React 生态下的 Three.js 封装）
  • 社区：
    • CSDN / 掘金 搜索 “WebGL 可视化”
    • GitHub Trending: topic:webgl

💡 高阶项目（作品集必备） ​

  1. 智慧城市 3D 大屏：集成楼宇、交通流、IoT 设备状态（可用假数据）
  2. WebGL + AI 可视化看板：如知识库 [1] 中提到的「3D 晋升看板」，用 ECharts-GL 渲染技能雷达
  3. 3D 网络拓扑图：节点关系用 Three.js 粒子线连接，支持拖拽和搜索

📅 学习节奏建议（12 个月计划） ​

时间重点
第 1-2 月基础准备（Canvas + 数学）
第 3–6 月原生 WebGL（WebGL Fundamentals 全学完）
第 7–10 月Three.js 核心 + 3 个项目
第 11-12 月专项深化 + 作品集打磨 + 投简历
